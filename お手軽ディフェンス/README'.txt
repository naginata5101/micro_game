"TowerDefense"

・初めに
このゲーム内の素材wikiよりパクらせて頂きました。
絵…'09贄田さん
音…'08尾澤さん


・目的
このゲームの名前からお分かりのように、
目的は「敵をゴールまでたどり着かせないこと」です。
敵のスタートは左端、ゴールは右端です。
敵は無限に湧いてくるので、頑張ってハイスコアを目指して下さい！！


・特徴
このタワーディフェンスは味方を道にも配置できます！（宮岡さんの発想）
そして、敵は進路にいる味方に攻撃をしてきます。
敵は分かれ道において、極力"道を塞いでいる味方が少ない道"を選んできます。
		　　　次いで"敵が少ない道"を選んできます。

敵を倒すと、金を "100 + ( 敵レベル - 1 ) * 10" 落とします。
因みに、敵は赤いマスで倒した場合はスコアが2倍になったりします。
また、スコアは敵のレベルに比例します。

また、味方ユニット一体／2000の金が掛かります。
撤退させると幾らかの金が返ってきますが、当然赤字です。
帰ってくる金額はユニットの体力に比例します。
