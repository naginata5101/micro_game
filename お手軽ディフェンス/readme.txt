"お手軽ディフェンス"


・初めに
このゲームをプレイして頂き有難う御座います。
このゲームは"（製作者が）お手軽ディフェンス"です。


・目的
このゲームのはタワーディフェンスと呼ばれるものジャンルのもので、
目的は「敵をゴールまでたどり着かせないこと」です。
敵のスタートは左端、ゴールは右端です。
敵が通る道の周辺に味方を配置して、ゴールまでに倒して下さい。
敵は無限に湧いて来ます。頑張ってハイスコアを目指しましょう！！


・システム
このタワーディフェンスは味方を道にも配置できます。
味方を道に配置した場合、
その味方は足止めを食らっている敵から攻撃を受けることになります。

敵を倒すと、スコアと金を落とします。
敵は赤いマスで倒した場合はスコアが2倍になったりします。

また、味方ユニット一体／2000円が掛かります。
撤退させると幾らかの金が返ってきますが、当然赤字です。
帰ってくる金額はユニットの体力に比例します。
